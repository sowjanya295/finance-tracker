# Personal Finance Tracker 💸

A simple frontend web app to track personal income, expenses, and balance. Manage transactions and view spending patterns in a clean UI.

## Features
- 📊 Display balance, income, and expenses
- 💵 Add transactions with description and amount
- 🔄 Update balance automatically
- 📋 List transactions with categories (e.g., food, groceries)
- 🗑️ Remove transactions

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript (or React/Vite if applicable)

## Setup
1. Clone the repo: `git clone <repo-url>`
2. Open `index.html` in a browser to run the app.

## Usage
- Enter a **Description** and **Amount** (+ for income, - for expense).
- Click **Add Transaction** to update balance and list.
- Use the 'x' button to remove transactions.
